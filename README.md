GrammarReader
=============
A university project for "Programming language theory" class @ [DIB](http://www.uniba.it/ricerca/dipartimenti/informatica).

## Content
You'll find a:
* C-like/Pascal-like Lexer;
* DFA (deterministic finite automaton;
* Date-DFA and a GrammarReader
